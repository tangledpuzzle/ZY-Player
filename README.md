## ZY Player 资源播放器

1. 全平台支持. windows, mac, linux.
2. 12个视频源. 未来更新更多的视频源.
3. 新增历史播放记录, 并记录播放进度.
4. 新增分享功能. 一键分享海报图片.
5. 新增精简模式. 支持修改透明度.
6. 全新布局配色.
7. 新增多语言.
8. 下载功能 (最大资源网和OK资源网支持下载.)
9. 更详细的视频分类
10. 收藏夹同步更新视频追剧.(手动更新)
11. 后台自动更新
12. 全局快捷键
13. 支持演员名称搜索
14. ...

觉得软件不错的, 点击右上角 star 收藏关注一波呀~

#### 全局快捷键:

|          快捷键          | 说明       | 主界面 | 小窗口 |
| :----------------------: | ---------- | :----: | :----: |
| `⌘ + ➡`  或  `Ctrl + ➡` | 下一集     |   √    |   √    |
| `⌘ + ⬅`  或  `Ctrl + ⬅` | 上一集     |   √    |   √    |
| `⌘ + ⬆`  或  `Ctrl + ⬆` | 减少透明度 |        |   √    |
| `⌘ + ⬇`  或  `Ctrl + ⬇` | 增加透明度 |        |   √    |
